Welcome to JIDA


Web application that retrieves SNP information for a genomic region of interest in Homo sapiens and calculates specific summary statistics for specified populations.

About :

The user will be able to retrieve SNP information when providing the app :

  • RS ID
  • Gene name
  • Genomic coordinates (start and end)

    • The application will return the following information for each SNP:

  • Name (rs value)
  • Genomic Position
  • Genotype frequencies and Allele frequency (provided for each population separately)

If multiple SNPs are returned, the user will be able to select the population(s) and summary statistics of interest, and the application will calculate them and plot their distribution in sliding windows along the region.

The summary statistics available are:

  • Genetic diversity
  • Haplotype diversity
  • Test against Neutrality (Tajima’s D)
  • If multiple populations are selected, then population genetic variation (FST value) for each pair of populations will be reported.


How to Use :


Click on the 'SNP Browser' loacted on the navigation bar at the top right.

You will be directed to a new page called 'JIDA SNP Browser'

To find your SNP's of interest please follow one of the following :

  • RS ID
  • Gene name or alias
  • Start and end positions
    •       chromosome = (21) by default

    Please also select from the checkboxes :

  • The population(s) of interest
  • The summary statistics of interest

    • Please type in the window size for the statistical calculations :

  • The window size    (default = 3)

    •         please click the Submit button and wait for your results!

    Results page :

    The output will be summary statistics tables (if selected) with the option to view the graph in the Show as Graph button

    Two graph are produced per statistic :

  • Statistic value vs the populations selected
  • Statistics value sliding window value vs genomic position with colours for each population

    • The Download text file button will produce a text file with the summary statistics


    Below the statistics there is an SNP table which contains the following information per SNP :

    chromosome number, genomic position, RS value, population, reference allele, alternate allele, the allele and genotype frequencies



    Glossary


    Genotype frequency

    Total individuals with a given genotype divided by the total number of individuals in a population.

    Allele frequency

    Allele frequency is the relative frequency of an allele (variant of a gene) at a particular locus in a population, expressed as a fraction or percentage.

    Nucleotide Diversity

    A measure of genetic diversity, the average pairwise number of nucleotide differences per site in a chromosome in a population.

    Haplotype Diversity

    A measure of heterozygosity, measure of uniqueness of a haplotype in a population, probability that two randomly sampled alleles are different. Close to 1 (the alleles are different); close to 0 (the alleles are closer).

    Tajima’s D

    Neutrality test, Compares the average number of pairwise differences along the base pairs in the population sample with the total number of variant sites in that sample. Negative (observing less variation than expected, selection removing variation); positive (observing more variation than expected, variation which is favoured but distinct from each other).

    Hudson FST

    Measure of population differentiation due to genetic structure. Close to 1 (no overlap in allele frequencies between the two populations); 0 (complete similarity between both populations); negative (there is more variation within than between the population)
